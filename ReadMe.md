# Quick start
```bash
./start.sh
```

## Для разработки (postgres, kafka, zoo и  keycloak)
```bash
./start.sh -dev
```

## Для остановки всех докер контейнеров
```bash
./start.sh -down-all
```

### Документация к api находится по адресу: </br>
http://localhost/api/v1/swagger-ui/index.html
